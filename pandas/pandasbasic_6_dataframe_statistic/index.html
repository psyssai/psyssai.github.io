6.Pandas DataFrame 통계

DataFrame의 주요한 통계 함수는 아래와 같다. - Describe : 전체 주요 통계량 출력 - sum : 합계 출력 - cumsum : 누적합계 출력 - mean : 평균값 출력 - Std : 표준 편차 출력 - var : 분산 출력 - median : 중앙값 출력 - count : 데이터 개수 출력 - min, max, idxmin, idxmax : 최소값,최대값,최소값의 index, 최대값의 index 출력 - corr, cov : 상관계수, 공분산 출력

위의 통계 함수를 비롯한 대부분의 함수에서 axis 옵션을 이용해 사용할 축을 설정할 수 있다. - axis = 0 : 열방향으로 통계량 산출(Default) - axis = 1 : 행방향으로 통계량 산출
